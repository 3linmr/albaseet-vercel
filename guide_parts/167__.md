# طلب حجز بضاعة
شاشة إنشاء وإدارة طلبات حجز البضائع للعملاء.
- إنشاء طلبات حجز جديدة
- تحديد البضائع المطلوب حجزها
- تحديد مدة الحجز
- ربط الطلب بالعميل
- تتبع حالة طلب الحجز
- تفاصيل البضائع المطلوب حجزها
- تاريخ انتهاء الحجز
- ملاحظات الطلب
تستخدم لإنشاء طلبات حجز البضائع للعملاء قبل البيع الفعلي.
