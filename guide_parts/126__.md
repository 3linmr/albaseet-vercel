# بيان حجز بضاعة
شاشة إنشاء وإدارة بيانات حجز البضائع للعملاء.
- إنشاء بيانات حجز جديدة
- ربط بيان الحجز بطلب الحجز
- تحديد الكميات المحجوزة
- تتبع حالة الحجز
- طباعة بيان الحجز
- رقم طلب الحجز
- تفاصيل البضائع المحجوزة
- تاريخ الحجز
- مدة الحجز
تستخدم لتوثيق حجز البضائع للعملاء حسب طلبات الحجز.
